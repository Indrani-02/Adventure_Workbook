# 📊 Adventure_Workbook (Data Analytics Project)

Welcome to **Adventure_Workbook** — a data analytics project designed to transform raw adventure and travel data into meaningful insights using **Excel, SQL, Power BI, and Tableau**.

This project demonstrates end-to-end data analysis, from data cleaning to visualization, making it ideal for aspiring and professional data analysts.

---

## 📖 Project Overview

Adventure_Workbook focuses on analyzing travel and adventure-related data such as trips, destinations, expenses, and activities. The goal is to uncover patterns, trends, and insights that help in better planning and decision-making.

---

## 🎯 Objectives

* Analyze travel behavior and patterns
* Track and optimize trip expenses
* Identify popular destinations and activities
* Build interactive dashboards for insights
* Demonstrate data analytics workflow using industry tools

---

## 🛠️ Tools & Technologies

* 📗 **Excel** – Data cleaning, preprocessing, and exploratory analysis
* 🗄️ **SQL** – Data querying, transformation, and aggregation
* 📊 **Power BI** – Interactive dashboards and reports
* 📈 **Tableau** – Advanced data visualization

## 🔄 Workflow

1. **Data Collection**

   * Gather raw travel/adventure datasets

2. **Data Cleaning (Excel)**

   * Remove duplicates
   * Handle missing values
   * Standardize formats

3. **Data Transformation (SQL)**

   * Create structured tables
   * Perform joins and aggregations
   * Generate analytical datasets

4. **Data Visualization**

   * Build dashboards in Power BI & Tableau
   * Create KPIs and visual insights

## 🚀 Getting Started

### Prerequisites

* Microsoft Excel
* SQL (MySQL / PostgreSQL / SQL Server)
* Power BI Desktop
* Tableau Desktop / Tableau Public

---

### Steps to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Adventure_Workbook.git
```

2. Open datasets from `/data/raw_data/`

3. Perform cleaning using Excel files

4. Run SQL queries from `/sql/queries.sql`

5. Open dashboards:

   * Power BI → `.pbix` file
   * Tableau → `.twbx` file
